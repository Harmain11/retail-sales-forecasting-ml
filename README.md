# Retail Sales Forecasting ML

## Overview
This notebook demonstrates retail sales forecasting using various machine learning models and deep learning (LSTM). It includes data preprocessing, feature engineering, visualization, model training, and performance comparison.

## Features
- Data download and organization from Kaggle dataset
- Exploratory data analysis with visualizations
- Time-based and rolling window feature engineering
- Implementation of Linear Regression, Random Forest, XGBoost, and LSTM models
- Model evaluation using R², MSE, RMSE, and MAE metrics
- Performance and efficiency comparison of models
- Clear, modular workflow optimized for reproducibility

## Tech Stack
- Python 3
- Jupyter Notebook / Google Colab
- Libraries: pandas, numpy, sklearn, matplotlib, seaborn, xgboost, tensorflow (Keras)

## How to Use
1. Clone the repository.
2. Open the notebook `sales-forecast-retail.ipynb` in Jupyter or Google Colab.
3. Run all cells sequentially to download data, preprocess, engineer features, train models, and visualize results.
4. Modify parameters or models as needed for experimentation.
